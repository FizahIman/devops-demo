# 🚀 DevOps Demo Project


## 🌐 Live Website

## 📌 What This Project Does
A complete CI/CD pipeline that automatically deploys a live website every time code is pushed to GitHub — no manual steps needed.

## 🔄 Pipeline Flow
## 🛠️ Tools & Technologies Used
| Tool | Purpose |
|---|---|
| **GitHub Actions** | CI/CD Pipeline automation |
| **Python** | Application code |
| **Ubuntu Linux** | Cloud runner environment |
| **GitHub Pages** | Live website hosting |
| **Git** | Version control |

## ⚙️ How It Works
1. Code is pushed to GitHub
2. GitHub Actions automatically triggers the pipeline
3. A fresh Ubuntu Linux server is created in the cloud
4. Python runs and generates the webpage
5. Website is automatically deployed live
